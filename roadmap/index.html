MMM is a modular DeFi protocol built for Monad.

The system combines a taxed ERC-20 token, custody vaults, deterministic deployment manifests, reward gating logic, multisig governance, and optional NFT-based reward boosts — all designed for auditability and operational discipline.

Protocol Architecture

Production deployment consists of independent modules with strict ownership boundaries and one-time wiring flows.

MMMToken
  • Buy / sell tax enforcement
  • Router + pair wiring
  • Trading lock + launch control
  • Tax exemptions
TaxVault
  • Receives MMM tax
  • Controlled reward distribution
  • Wired once to other vaults
RewardVault
  • Minimum hold time
  • Claim cooldown
  • Minimum balance requirement
  • Excluded supply rule
SwapVault
  • Router-based swap execution
  • Controlled by TaxVault
  • DEX isolation layer
Multisig Vaults
  • MarketingVault (2-of-3)
  • TeamVestingVault (2-of-3)
  • USDC custody
BoostNFT (Optional)
  • Reward weight multiplier
  • Tiered NFT system
  • Must remain bounded + auditable
